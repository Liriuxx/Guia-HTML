Css

Es un lenguaje basado en reglas las cuales determinaran el estilo que se aplicaran a un elemento o un grupo de elementos.

Existen 3 opciones para escribir el conjunto de reglas en css

  1. Estilos de línea

    2. El estilo se añade directo a la etiqueta de apertura del elemento html

    Curso CSS

  2. Etiqueta style

    3. Se añade en el "head" y dentro de style se escribe el estilo, este se aplicara a todas las etiquetas que seleccionemos, por ejemplo todas las h3, h2, etc


    La sintaxis es la siguiente:

    head

    style

    h2 {

    color: blue

    }

    /style

  3. Archivo .css

    4. Se crea un archivo de tipo .css, el cúal se va a vincular al archivo html, esta opción es mucho más recomendada

    Para vincular el archivo html al css, se emplea la etiqueta link href="archivo conexión" rel="stylesheet"(Esto indica que es una hoja de estilo)

Selectores

Existen diferentes tipos de selectores

  • Selector de tipo

    • Selecciona todos los elementos que coinciden con el nombre del elemento especificado

    Por ejemplo, si escribirmo h3, se aplicara a todos los elementos h3:

  • selector de clase

    • Selecciona todos los elementos que tienen el atributo "class" especificado

    Para establecer la clase, simplemente se añade después de la etiqueta como "class="Nombre_de_clase"

    Posteriormente, en el archivo .css, se va a referenciar con un "." seguido de lnombre de clase

    Por Ejemplo:

    ul class="Lista_de_accesorios"

    En .css se víncula:

    .Lista_de_accesorios {

    color:chocolate

    }

  • Selector de ID
  • Selector universal

    • Selecciona todos los elementos, se emplea solo un *, y se va a aplicar a todo lo que contenga mi página

  • Selector de atributo

Edición del tamaño de fuente

Para editar el tamaño se emplea "font-size: 00px;

px refiere a los pixeles

Si vamos a aplicar un mismo estilo a diferentes etiquetas, podemos simplemente separar por "," cada etiqueta, y debe ir en renglones separados, y al finalizar se indica el estilo, por ejemplo:

h2,

ul,

ol {

font-size: 20px;

}

Con esto se aplicaria el mismo tamaño de fuente a 3 etiquetas.

Para emplear distintas fuentes, vscode tiene predefinidas algunas familias, las cuales accedemos con las etiquetas "font-family", pero en caso de requerir una, podemos acceder a "fonts.google.com, ahí buscar la fuente deseada y seleccionar el apartado de "view selected families", copiar el "link" y las "CSS rules" en el head del html, y para css las "rules" en "font-family"

Aplicar css medinate id

Para vincular los cambios mediante id, empleamos el símbolo #, esto le indica a css que se trata del id, por ejemplo: #Aqua.

Padding

El padding es la distancia que hay entre el borde de un elemento html y su contenido

Padding a cada lado

Se puede ajustar cada lado del padding, simplemente cambiando a -top,left,right,bottom

Margin

Es la distancia entre elementos

Se emplea el comando "margin:"

Se puede personalizar por cada lado, similar al padding, margin-top, margin-bottom-, margin-right, margin-left, de la misma manera que el padding, se puede modificar en una sola línea, considerando el sentido del reloj para acomodar, por ejemplo

margin top right bottom left;
