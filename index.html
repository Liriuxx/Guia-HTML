Todos los elementos del contenido principal de la página deben encontrarse dentro de la etiqueta "main"

Es muy importante que los elementos se encuentren identificados por la identación (espacios)

El contenido de main, debe ser único, excluyendo cualquier contenido que se repita a través de un conjunto de documentos como barras laterales, enlaces de navegación, información de derechos de autor, logos del sitio y formulario de búsqueda (a menos que la función principal del documento se a un formulario de búsqueda

Main representa el contenido principal del body

No debe existir más de un elemento main y este no debe ser descendiente de un elemento "article, aside, footer, header o nav"

Para quitar espacios, seleccionamos todo el texto que queremos modificar, posteriormente presionamos shift + tabulador

    Formularios

    Para crear formulaarios, se emplea la etiqueta form, incluye un cierre, ademas, se deben ingresar etiquetas auxiliares input, con auxiliar Type=text, para indicar que es el ingreso de información

    La etiqueta button debe incluir un Type puede ser submit que es para enviar y placeholder para el texto auxiliar que describa la acción del botón

    Crear campos obligatorios en un formulario

    Para evitar el que un formulario no este completo, se agrega el atributo "required"

    Botones de radio

    Un elemento de tipo radio, crea un boton de radio o de opción, que es aquel en el que solo se selecciona una opción

    Hola

    Atributo Label

    El atributo "label" ayuda a que cuando tocamos el texto se acciona el botón de radio

    Relacionar etiqueta con el boton

    A la etiqueta se le puede agregar un atributo "for" para relacionar el boton con la etiqueta, y así halla correspondencia

    Para que solo se marque una opción en los botones de radio, se debe crear un grupo de botones de radio, para hacerlo se emplea el atributo "name" en el atributo input, esto para cada botón

    Para hacer que los botones de radio esten en líneas separadas, debemos ingresar la etiqueta "br" (break), el cual representa un salto de linea
